# IM
Project
这个是我跟一个朋友一起写的一个聊天小软件
界面是仿的微信，用python的多线程+Tkinter+socket+Mysql数据库实现的
还有很多功能没有完善，现在只写了登陆，注册，添加好友，私聊，在线群聊，聊天记录几个功能模块

有兴趣的伙伴可以了解下
